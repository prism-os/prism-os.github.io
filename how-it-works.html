How it works

To use Prism OS, the board needs to be flashed for the first time using prism command line tool, after which all future updates and installs can happen directly without any computer. Prism OS requires boards that have Wi-Fi connectivity like esp8266, esp32 etc.

We are planning to release a reference unit through crowdfunding platforms or otherwise that would be ready to go out of the box and straight forward for general consumers.

After first flash, users can download and install the Prism companion app available for both Android and iOS platform, after which the phone app can be used to install apps, software updates and even control the device directly. Once paired and signed in, users can install apps onto the device from anywhere in the world directly from Prism App Store's web app!

Check out the documentation to know more and get started!
